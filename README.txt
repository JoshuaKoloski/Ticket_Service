In order to build the solution and run the tests you must navigate to the name of the 
project "Ticket Service" in the terminal and 
then run the following commands in the following order
1) gradle assemble
2) gradle build

Gradle build will automatically run the test cases. You can see the results
of the test cases by navigating to "TicketService\build\reports\tests\test\classes".
Double click on the html files and you can view the results on a web page.
